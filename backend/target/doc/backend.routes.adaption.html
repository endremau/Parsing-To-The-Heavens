backend.routes.adaption

adaption-request-handler

(adaption-request-handler {obj :body, :as req})

HTTP GET handler for requesting network adaptions. This endpoint will log the data it recieves and respond with a suitable adaption. The endpoint accepts a json body with a device-id integer and a netcoll key NetJSON NetworkCollection object which has to be a collection of other NetworkCollection objects
